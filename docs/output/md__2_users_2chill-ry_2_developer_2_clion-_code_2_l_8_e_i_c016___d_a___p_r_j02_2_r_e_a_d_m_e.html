TSP - Traveling Salesman Problem

Description

This project tries to find the best and feasible algorithm to solve the Traveling Salesman Problem (TSP). The TSP is a well-known problem in the field of combinatorial optimization. The problem consists of finding the shortest possible route that visits each city exactly once and returns to the origin city. The problem is NP-hard, which means that there is no known algorithm that can solve the problem in polynomial time.
